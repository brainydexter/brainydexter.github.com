wordsAway Gameplay from Priyank jain
Words-Away

Words-Away is a 2D game which is a cross between Tetris and Boggle. The objective of the game is to place the falling letters adjacent to each other and form words to clear them from the board. This is further complimented by different powerups and score multipliers.

I prototyped the game as a sideproject in ThreeJs in about twenty days. The prototype was well received by friends and family who motivated me to publish a polished game on the app store.

I chose to use/learn Unity in the process. Target platform being mobile forced me to be extra cautious about performance. I used this opportunity to write a lot of boiler plate code/unity components that I could use in the future as well. Some of the components are

  • Event Aggregator following a publisher/subscriber model
  • Trie for fast lookup and optimizing it for space
  • Object pooling/recycling without Garbage Collection
  • Exposing components for designers
  • Letter Generation using weighted probability
  • Conveyor belts
I got the opportunity to work with some extremely talented artists and I'm proud of building this game!

Time Duration 60 days (as side-project)
Technology C# / Python
Graphics API Unity v5
Version Control Perforce
Prototype Three JS src

Kolor

A 3D first-person genre game in which I have introduced a unique game mechanic. The objective of the game is to claim other players by coloring them in your own color and once you own more than 70% of the players, you are victorious.

I prototyped the game in C#-XNA in about ten days. XNA hides a lot of gory details from the developer and bare-basics is what I really wanted to learn and implement, which motivated me to make this game entirely in C++/OpenGL. To this extent, I chose not to use any rendering or a physics engine. Following are some of the things I implemented:

  • Collada DAE importer
  • Frame Transformation classes
  • First person camera
  • User-Input classes
  • Broad/Narrow phase Collision Detection
  • Bounding-Sphere generation


Technology C++ / Qt Framework
Graphics API OpenGL
Libraries OpenGL Math, Boost
IDE Microsoft Visual Studio 2008 Professional
Version Control SVN
